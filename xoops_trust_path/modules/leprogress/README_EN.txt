Leprogress
==========
Progress Management Module.

Requirement
-----------
-XOOPS Cube Legacy 2.2

Main Feature
------------
-Approval Persons Setting with review order.
-Approval reviews the article in sequence.
-Approval do approve/reject the article and comment to it.
-After all of approval has finished to review, the article's status is set 'finished' and call delegate function for the original module.
